# Technology Stack

## Frontend Framework
- **React 18** (^18.3.1) - UI component library
- **TypeScript 5.8** - Static typing
- **Vite 5.4** - Build tool and dev server (port 8080)
- **React Router 6.30** - Client-side routing

## UI & Styling
- **TailwindCSS 3.4** - Utility-first CSS framework
- **Radix UI** - Unstyled, accessible component primitives
- **shadcn/ui** - Radix UI + TailwindCSS components
- **Framer Motion 12.38** - Animation library
- **Lucide React** - Icon library
- **Next Themes** - Dark mode support

## State Management & Data
- **@tanstack/react-query 5.83** - Server state management
- **Supabase JS SDK 2.101** - PostgreSQL backend + auth
- **React Hook Form 7.61** - Form handling
- **Zod 3.25** - Schema validation

## Additional Libraries
- **Sonner** - Toast notifications
- **React Resizable Panels** - Resizable UI panels
- **QRCode React** - QR code generation
- **Recharts** - Data visualization
- **Date-fns 3.6** - Date utilities
- **Embla Carousel** - Carousel component
- **CVA** - Component variant automation
- **CLSX/Tailwind Merge** - Conditional CSS classes

## Backend Integration
- **Supabase** - PostgreSQL, Auth, Real-time
- **Lovable Cloud Auth** - OAuth authentication
- **Bridge Server** - Node.js server handling Twilio-Gemini WebSocket

## Development Tools
- **ESLint 9.32** - Code linting
  - React Hooks plugin
  - React Refresh plugin
  - TypeScript ESLint support
- **Vitest 3.2** - Unit testing framework
- **Playwright 1.57** - E2E testing
- **JSDOM 20** - DOM simulation for tests
- **Testing Library** - Component testing utilities

## Mobile (Capacitor)
- **@capacitor/core** - Cross-platform mobile framework
- iOS and Android support
- Plugin system for native features

## Development Environment
- **Node.js** ecosystem
- **macOS** (Darwin) compatible
- Lovable-tagger for component metadata
