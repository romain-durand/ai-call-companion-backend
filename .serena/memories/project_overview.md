# AI Call Companion - Project Overview

## Purpose
AI-assisted call handling and contact management application. A React-based web and mobile app that helps users manage calls, contacts, and interactions with AI assistance.

## Key Features
- Call handling and history tracking
- Contact/profile management
- Calendar integration
- AI assistant for call support
- Activity monitoring
- Settings and personalization
- Mobile and web support

## Project Context
- Built with Lovable.dev framework
- Supports both web (Vite SPA) and mobile (Capacitor iOS/Android)
- Real-time data sync with Supabase backend
- OAuth authentication integration
- Bridge server with Twilio and Gemini integration

## Development Status
Active development with recent UI updates and feature refinements. The project uses component-based architecture with TypeScript for type safety.
